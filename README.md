# 📖 Bookan - Social Book Discovery & Management Platform

**Bookan** is a comprehensive web application developed for book enthusiasts. It allows users to manage their personal libraries, share inspiring quotes, and discover new books through a social interface.

## ✨ Key Features
- **📚 Library Management:** Add, edit, and list books with detailed information (Author, Category, Publisher, etc.).
- **💬 Quote Sharing:** Share impactful quotes from books and browse what others have shared.
- **❤️ Favorites System:** Add your favorite books and quotes to a personal list for quick access.
- **👥 Social Interaction:** Follow other readers and stay updated with their literary journey.
- **🔍 Dynamic Search:** Easily find specific books or quotes within the platform.

## 🛠 Tech Stack
- **Backend:** Pure PHP
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** MySQL
- **Environment:** AppServ / XAMPP
